# 📘 🤖 AI SYSTEM – SETUP & RUN GUIDE

## 🔹 1. Requirements

Make sure your system has:

```bash
Python 3.10 or above
```

Check version:

```bash
python --version
```

---

## 🔹 2. Install Required Libraries

Run this command:

```bash
pip install transformers torch requests
```
---

## 🔹 3. (Optional but Recommended)

Install faster backend:

```bash
pip install accelerate
```

---

## 🔹 4. Project Structure

Example:

```text
project/
│── dataset.py
│── tokens.py
│── prompting.py
│── transformer.py
│── llm.py
│── fine_tuning.py
│── reinforcement.py
│── agents.py
│── ai-data-analyst-project.zip
│── README.md
```

---

## 🔹 5. Run Any Program

Use:

```bash
python filename.py
```

👉 Example:

```bash
python dataset.py
python agents.py
```

---

## 🔹 6. First Run (Important ⚠️)

* Model will download automatically
* Internet required **only first time**
* After that → works offline

---
